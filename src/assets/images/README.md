# Assets - Images

This directory contains all image assets for the Craftify app.

## Required Images

### App Icons
- `icon.png` - App icon (1024x1024px)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024px)
- `favicon.png` - Web favicon (192x192px)
- `splash.png` - Splash screen image (1242x2436px)

### Placeholder Images
- `placeholder-template.png` - Template placeholder
- `placeholder-user.png` - User profile placeholder
- `placeholder-image.png` - Image element placeholder

### Onboarding
- `onboarding-1.png` - First onboarding slide
- `onboarding-2.png` - Second onboarding slide
- `onboarding-3.png` - Third onboarding slide

### Feature Icons
- `premium-badge.png` - Premium feature indicator
- `watermark-logo.png` - Watermark logo for free exports

## Image Guidelines

- Use PNG format for images with transparency
- Use JPG format for photos and images without transparency
- Optimize all images before adding to the project
- Use WebP format for better compression (when supported)
- Keep file sizes as small as possible while maintaining quality
- Use descriptive, lowercase, hyphenated filenames

## Image Optimization Tools

- TinyPNG: https://tinypng.com/
- ImageOptim: https://imageoptim.com/
- Squoosh: https://squoosh.app/
